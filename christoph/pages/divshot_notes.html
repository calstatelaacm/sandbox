Divshot Notes

Things to Install:

  • git
  • node.js
  • https://divshot.com/

Creating and Uploading a Site:

  1. From your command line (Terminal on Mac OS, Command Prompt on Windows), navigate to your website folder using the "cd" command.
  2. Once inside your folder, use the "divshot init" command in the command line. Make sure you have an index.html file in the root directory.
  3. You will be prompted to enter a name. This will be part of your url (your full url will be "yoursitename.divshot.io")
  4. Once your divshot app has been successfully created, use the "divshot push" command to publish your site.
    • note: by default, "divshot push" hosts your site at "development.yoursitename.divshot.io". when you're ready for the world to see it, use the "divshot push production" command in your command line.
